# Project Context

**Project Name:** PenguinPortfolio
**Description:** A portfolio web application.
**Tech Stack:** HTML, CSS, JavaScript, TailwindCSS, npm.

## General Rules
- Always use vanilla JavaScript where possible, unless another framework is explicitly requested.
- Prioritize clean, modern, and responsive web design.
- Format code using standard best practices.

## Architecture & Guidelines
- This is a standard frontend web project.
- The `index.html` file serves as the main entry point.
- The `styles` directory and `tailwind.config.js` handle the styling configuration.
- Any JavaScript logic is placed in `script.js` or organized into modules if complexity grows.
- Use `npm run ...` for building and development scripts as defined in `package.json`.
